# HarmonyOS应用包

这是通达信问答鸿蒙应用的HAP包。

## 文件信息

- **文件名**: entry-default-signed.hap
- **版本**: 1.0.0
- **构建时间**: 自动生成
- **用途**: 测试安装流程

## 安装方法

### 方法1: 使用hdc命令

```bash
hdc install entry-default-signed.hap
```

### 方法2: 直接安装

1. 将HAP文件复制到MiniPad
2. 在MiniPad上打开文件管理器
3. 点击HAP文件安装
